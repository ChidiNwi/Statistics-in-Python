# Statistics in Python
This project demonstrates statistical analysis using Python. The goal of this analysis was to asssess pump performance among the water treatment plant. 
This project uses the following tools and libraries:
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook
- These tools are commonly used for data exploration, statistical analysis, and visualization in Python.
The analysis follows a standard data analysis workflow:
Import Python libraries
Load and inspect the dataset
Perform data cleaning and preparation
Conduct descriptive statistical analysis
Visualize distributions and relationships
Interpret insights from the data
